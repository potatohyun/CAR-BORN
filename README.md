## 프로젝트 소개
**CAR-BORN**은 블록체인 프로젝트로, 가상의 중고차 거래 플랫폼을 위한 시스템 설계 및 구현을 하였으며, 어떤 기능이 필요할지 요구사항을 추려내고, 해당 요구사항에 따라 기능을 구현한 **개발진의 역량향상** 프로젝트입니다.

크게 웹 Front-end, Back-end, Blockchain 파트로 나누어, 7주동안 프로젝트를 진행했습니다.
